Challenge: build a classic hacker terminal. It must include the following pieces!

Black background with green text

An input area where you can access the following commands:

  • list- lists all supported commands
  • connect- shows a fancy progress bar that imitates connecting to a remote server
  • engage- vertically moving letter cover the background
  • disEngage- changes background back to a static black
  • serverCorrupt- displays a variety of errors ending with a "hack complete" message
  • engageHack- shows a constant never ending flow of random letter/numbers popping up
